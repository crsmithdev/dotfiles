---
description: Commit and push changes to GitHub
---

1. Run `~/.claude/scripts/ship.sh` to review changes
2. Draft a terse commit message following the repo's style (details in subject, body only if >100 chars)
3. Stage relevant files with `git add`
4. Commit with the message (include the standard footer)
5. Push to origin
6. Report the commit hash and push status
